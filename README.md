  - Copying Nvim-plugins
    ```bash
    mv ~/.local/share/nvim/Lazy ~/.local/share/nvim/Lazy.bak # backup original configs
    cp nvim-plugins -r ~/.local/share/nvim/Lazy # Copy the new plugins 
    ```
